# Sistema CRUD de Estudiantes con SQLite (MVC)

## Descripción

Este proyecto implementa un sistema CRUD para la gestión de estudiantes usando Python y SQLite. Permite crear, consultar, actualizar y eliminar registros desde consola.

El código fue reorganizado utilizando el patrón MVC para separar responsabilidades y mejorar la organización.


## Estructura del proyecto

│

├── main.py

│

├── models/

│ └── estudiante.py

│
├── views/

│ └── estudiante_view.py

│

├── controllers/

│ └── estudiante_controller.py

│

├── database/

│ └── conexion.py

│

└── data/

└── crud.py



## Ejecución

1. Clonar el repositorio:
  git clone

2. Entrar a la carpeta:
  cd gestor_estudiantes_mvc

3. Ejecutar:
  python main.py


## Funcionalidades

- Crear estudiante  
- Listar estudiantes  
- Actualizar nota  
- Eliminar estudiante
